# PWA Icons

This directory should contain the following icons for PWA support:

- `icon-192.png` - 192x192 pixels, standard icon
- `icon-512.png` - 512x512 pixels, standard icon
- `maskable-icon.png` - 512x512 pixels, maskable icon for Android

## Icon Requirements

### Standard Icons

- Must be PNG format
- Should have a transparent or solid background
- Should be recognizable at small sizes

### Maskable Icon

- Must be PNG format
- Must have a solid background (no transparency)
- Safe area should be centered (content within 80% of the icon)
- Used for adaptive icons on Android

## Creating Icons

You can create these icons using:

- Figma, Sketch, or Adobe Illustrator
- Online tools like RealFaviconGenerator or PWA Asset Generator
- ImageMagick for command-line generation
